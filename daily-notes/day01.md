# Day 1 – Introduction to Excel & Basic Formula

## What is Excel?

Microsoft Excel is a spreadsheet software used to store, organize and analyze data.

It is widely used for:

- Data analysis
- Financial calculations
- Business reporting
- Data visualization

## Excel Structure

Excel organizes data using:

Rows → Horizontal lines numbered 1,2,3  
Columns → Vertical sections labeled A,B,C  
Cells → Intersection of a row and column

Example cell: B3

## First Formula

Excel formulas always start with =

Example:

=A1 + A2

If:
A1 = 5
A2 = 10

Result: 15

## SUM Function

The SUM function adds numbers.

Example:

=SUM(A1:A5)

This adds all numbers from A1 to A5.

## Key Takeaways

- Excel uses rows, columns and cells
- Formulas start with =
- SUM is one of the most basic functions
